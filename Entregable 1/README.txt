Para correr el programa, se debe incluir en la misma carpeta que este archivo, un archivo csv llamado 'trivia_questions.csv'
Es necesario instalarse la librería pandas. Para hacerlo, se precisa abrir la terminal y correr 'pip install pandas'.
Luego de ser instalado, simplemente correr el programa.

Recursos implementados:
-Librería pandas
-Decoradores/Wrappers
-Map
-Reduce
-Sample
-Funciones Lambda
-Iterables
-itertools.chain


De haber tenido mas tiempo, nos hubiera gustado implementar las siguientes caracteristicas:
-Monadas: utilizarlas para contar y mostrar el score
-Generadores: utilizarlas para poder iterar las preguntas una a una, pero era necesario un for, por lo que omitimos utilizarla de esta manera
